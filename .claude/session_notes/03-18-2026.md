### 3. Print size change: 7.5×10 → 8.5×11
- Updated `@page` margin from `0.5in` → `0`
- Updated print cert dimensions from `7.5in × 10in` → `8.5in × 11in`
- Updated screen CSS variables `--w`/`--h` from `720px × 960px` → `816px × 1056px`
- Updated JS `scaleCert()` hardcoded dimensions to match
- Increased design margins (borders, corners, content inset) to compensate for loss of page margin

### 4. Print Size Toggle (8×10 vs 8.5×11)
- Added **Print Size** toggle to toolbar with two buttons: `8 × 10` and `8.5 × 11`
- `SIZE_MODES` JS object holds screen dimensions and print parameters for each mode
- `setMode(mode)` function:
  - Updates `--w`/`--h` CSS variables on `<html>`
  - Dynamically rewrites `<style id="print-size">` with the correct `@page` rule and cert print dimensions (only reliable way to swap `@page` at runtime)
  - Calls `scaleCert()` and `adjustSpacing()`
  - Saves selection to `localStorage`
- Mode restored on page load via `loadFromStorage()`
- **8×10 mode** centers cert on 8.5×11 paper with `0.5in` top/bottom, `0.25in` side margins — sized for mat-framed presentation

### 5. Bug fixes (size toggle)
- **Background disappearing**: Card stock color picker used `querySelectorAll('.color-btn')` which matched the size toggle buttons too (they share the class for styling). Clicking a size button triggered the card stock handler and set `--cream` to `"undefined"` (invalid color), wiping the certificate background. Fixed by scoping selector to `.color-btn[data-bg]`.
- **Bottom section overflowing borders**: `adjustSpacing()` was not called after `setMode()`, leaving stale margin values from the old canvas height. Fixed by calling `adjustSpacing()` at the end of `setMode()`.

## Key Architecture Reminders
- `SIZE_MODES` and `setMode()` in JS control all size-dependent behavior
- `<style id="print-size">` is dynamically written — do not put `@page` rules in the static `<style>` block
- Card stock color picker must use `.color-btn[data-bg]` selector — size buttons share `.color-btn` class
- `adjustSpacing()` must be called any time the cert canvas dimensions change
